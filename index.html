Tricky dungeon

Ce projet a été réalisé dans le cadre de projet de fin d'année de Terminale dans la matière "numérique et science informatique" en 3 semaines à deux personnes. C'est un petit jeu d'exploration de donjon généré procéduralement avec des monstres dans chaque salle. Après avoir éliminé tous les monstres, il faut trouver la bougie pour faire apparaître le boss du donjon et passer au niveau suivant. Le projet a été entièrement réalisé en python en utilisant la bibliothèque graphique pygame. Mon rôle dans ce projet a été de faire l'algorithme de génération du donjon ainsi que gérer le rendu des éléments à l'écran.

Automate cellulaire des voisins de Moore

Ce projet est une implémentation d'un automate cellulaire, algorithme dans lequel on utilise des cellules qui intéragissent entre elles en fonction de leurs voisins et de règles. Le but de ce projet a été de découvrir les automates cellulaires pour la création de terrain dans un jeu. Ce projet a été réalisé en python en utilisant la librairie graphique pygame et a été réalisé au début de l'année 2022 en une après-midi comme expérimentation.

Analyse sur les accidents de la route

Le but de ce projet était de créer un rapport en anglais pour faire de la sensibilisation sur les accidents dans une zone en France d'une certaine population. Il a donc fallu extraire les données importantes pour le projet d'une base de données en open-data qui recense les accidents de tout type dans toute la France, nettoyer les données et les analyser avec des statistiques réalisées dans le langage R en utilisant la bibliothèque statistique tidyverse. Ce projet a été réalisé a deux pendant une situation d'apprentissage évaluée du deuxième semestre du bachelor universitaire technologique d'informatique.

Installation d'un service réseau

Pendant cette situation d'apprentissage évaluée, il a fallu installer Debian Bullseye sur une machine virtuelle dans le but d'une utilisation comme serveur web/base de données. Après l'installation du système, il a fallue installer et configurer Apache pour ajouter PHP et un outils de gestion de base de donnés (phpPgAdmin). A la suite de cela il a aussi fallu installer et configurer PostgresSQL pour accepter des connections extérieures, créer des utilisateurs, créer une base de données etc... Le rendu de ce projet était un document en anglais expliquant comment faire ce type d'installation.
