Détails :

Ce projet à été réalisé à 7 dont 2 personnes concentrées sur l'administration et 5 autres personnes, dont moi-même, sur le développement du projet. Le sujet était de créer un site web permetant à des lycéens d'apprendre l'histoire de la méditerranée pendant l'antiquité. Le temps de réalisation a été de 3 semaines. Nous avons utilisé du PHP pour le back-end et, pour le développement du jeu, JavaScript en utilisant le moteur de jeu Phaser.
Nous avons développé, en autre, un système d'authentification pour les utilisateurs, un système de messagerie entre les utilisateurs ainsi qu'une sauvegarde de l'avancement dans le jeu pour les utilisateurs.
