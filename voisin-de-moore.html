Automate cellulaire des voisins de moore

Projet sur github

Compétance technique

  • visual studio code
  • pygame
  • algorithmie

Compétence transversale

  • recherche

Ce projet est une implémentation d'un automate céllulaire, algorithme ou on utilise des céllules qui intéragisse entre elle en fonction de leurs voisins et de règles. Le but de ce projet a été de découvrire les automates céllulaire pour la création de terrain pour un jeu. Ce projet a été réalisé en python en utilisant la librairie graphique pygame et a été réalisé au début de l'année 2022 en une après-midi comme expérimentation.
