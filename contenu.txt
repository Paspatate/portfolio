Sae1.01 :
Titre : 				        programmation et optimisation d’un algorithme qui détèrmine du sujet d’un article.
compétence technique :          java, intelIJ, algorithmie
compétence transversale :       travail en collaboration, anglais, 
présentation :

Ce projet a été réalisé dans le cadre d’une situation d’apprentissage évalué qui s’appelle
« Implémentation d’un besoin client et comparaison d’approche algorithmique ». Le but de ce projet a été de créé un algorithme qui,
grâce a un dictionnaire de mot significatif, permet de déterminé le sujet d’un article dans les catégories : 
- politique
- sport 
- science
- économie
- culture

Les mots significatif sont choisie a l'aide de 500 autre article. 
L'autre but de cette SAE est d'optimisé l'algorithme de détèrminaison de catégorie.
Ce projet a été réalisé a deux en java pure dans intelIJ avec un compte rendu en anglais.

présentation:

Ce projet a été réalisé dans le cadre de projet de fin d'année de Terminale dans la matière numérique est science informatique en 3 semaine a deux personne. 
C'est un petit jeu d'exploration de donjon généré procéduralement avec des monstres dans chaque salle. Après avoir éliminé tous les monstres, il faut trouver la

Mon rôle dans ce projet a été de faire l'algorithme de génération du donjon ansi que géré le rendu des éléments a l'écran.

Sae2.04
Titre:                          Exploitation d'une base de donnée
compétence technique:           PostgresSQL, R, tidyverse
compétence transversale:        anglais, collaboration, gestion de projet
présentation: 

Le but de ce projet était de créé un rapport en anglais pour faire de la sensibilisation sur les accidents dans une zone en France d'une certaine population.
Il a donc fallu extraire les données importante pour le projet d'une base de donnée en open-data qui ressence les accidents de tous type dans toute la France,
néttoyer les donnés et les analysés avec des statistiques faite dans le language R en utilisant la bibliothèque statistique tidyverse.
Ce projet a été réalisé a deux pendant une situation d'apprentissage évalué du deuxième semestre du bachelore universitaire technologique d'informatique.
